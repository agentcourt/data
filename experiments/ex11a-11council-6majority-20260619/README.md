# ex11a, 11-Member Council

## Scope

This experiment contains five direct local AgentCourt arbitration runs for `arb/examples/ex11a`.  Each run used an 11-member council, a six-vote threshold, and the sampled council pool retained in the run artifacts.  The directories are numbered in chronological order from `run-01` through `run-05` so the path names are stable and short.

The retained output preserves full run records, event streams, transcripts, evidence stores, council-turn files, logs, policy files, and local runner metadata.  Each run also retains `policy-11-of-6.json`, which records the explicit council policy used to start the run.  The source timestamp directory names are listed below for provenance.
